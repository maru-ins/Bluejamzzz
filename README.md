Bluejamzzz adalah repository proyek yang berisi firmware Arduino, komponen UI, dan struktur pendukung yang dikembangkan untuk perangkat berbasis **ESP32**.  
Repository ini digunakan sebagai pusat pengelolaan source code dan eksperimen pengembangan sistem embedded.
